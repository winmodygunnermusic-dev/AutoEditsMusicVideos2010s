# Tutorial: Use the Super Deluxe Auto-Editor on Windows 8.1

## Quick start
1. Open:
   ```bat
   python autoedit_gui.py
   ```
2. Add **video files** and/or **video folders**.
3. Add **music sources** (and optional **SFX sources**).
4. Set output filename.
5. Click **Run Auto-Edit**.

## Recommended long-video settings
- **Total clips**: 200 to 500+
- **Min clip sec**: 1.2
- **Max clip sec**: 5.0
- **Transition**: Fade 0.2 to 0.5 sec
- **Audio mode**: Combine all shuffled
- **Draft mode**: On for rough cuts, Off for final export

## Feature notes
- **Beat-aware edit**: Reads TBPM tag when present; uses BPM fallback otherwise.
- **Dance effects**: Raise Dance energy for stronger color/motion style.
- **Remix mode**: Nightcore, Slow Jam, Hyper Dance available.
- **Remix styles**: Chaos remix, Beat remix, Meme remix, YouTube Poop, TikTok, AMV.
- **Intro/Outro pools**: Add branded openers/enders.
- **Branding logo**: Overlay watermark/logo in final render.
- **Presets**: Save and load JSON templates for repeatable projects.
